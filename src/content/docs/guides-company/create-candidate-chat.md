---
title: "¿Cómo Creo Un Chat Con Un Candidato?"
description: "Aprende cómo iniciar un chat con un candidato"
---

## Paso 1: Seleccionar El Candidato

Presionamos el nombre del candidato con el que queremos iniciar un chat.

![Seleccionar candidato](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/e530beac9ab51aa41c1dcdc71e7fa1e3/screenshot/183dd723-5a2e-4193-aa64-914533049733.png)

## Paso 2: Presionar Message

Presionamos donde dice **Message (Mensaje)** para iniciar la conversación con el candidato.

![Presionar Message](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/4e9d26208d53616fdc769a5c85131489/screenshot/7b4eb3d2-3874-45bd-a710-a595dc025867.png)

## Paso 3: Chatear Con El Candidato

Ya podremos chatear con el candidato seleccionado. Se abrirá la ventana de chat donde podemos enviar mensajes y mantener la comunicación.

![Chatear con candidato](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/7ac8c5aee79f7302c9a8f62609bd60dc/screenshot/228195c2-3e50-4808-b2a8-ba02e7f1e9a9.png)
